网页不能替你启动 Python。这是浏览器安全策略：任意网站（含本地打开的 HTML）若能在点击时执行本机命令，恶意页面即可盗数据、删文件；因此不存在「在页面里点一下就自动跑 python …」的合法做法。必须先有已在跑的后端，本页的「拉取配置 / 一键生成」才会生效。 少自己动手的路径：使用本 skill 下 desktop/ 的 Electron 桌面壳（npm install 后 npm start）：窗口内可一键启动后端并内嵌本配置页，见 desktop/README.txt。或双击根目录 start_serve_web.bat；或在 Cursor：Tasks: Run Task → Arena: serve web (8765)。再用系统浏览器打开 http://127.0.0.1:8765/web/arena-setup.html 并点「从 serve_web 拉取最新配置」。 Simple Browser 多为纯静态：本页会读 web/api/arena-config.json 填表；一键生成仍依赖上面启动的后端。
