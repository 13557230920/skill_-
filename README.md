# skill_-

本仓库为 **金融 AI 擂台（Financial AI Arena）** 独立发布目录，代码在 **`financial-ai-arena/`**。

- **说明与用法**：[financial-ai-arena/README.md](financial-ai-arena/README.md)  
- **完整能力规格**：[financial-ai-arena/SKILL.md](financial-ai-arena/SKILL.md)  
- **上游开发路径**（若你本地是 monorepo）：`jingrong_skill/skills/financial-ai-arena/`

快速启动：

```bash
cd financial-ai-arena
pip install -r requirements.txt
python scripts/serve_web.py
```

浏览器打开 `http://127.0.0.1:8765/web/index.html`。请勿将 `.env` 或 API Key 提交到 Git。
