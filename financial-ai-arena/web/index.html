使用前说明（像素页 / serve_web）

擂台页为 纯 Canvas 像素绘制；数据来自 arena_state.json。请先在 skill 根目录执行 python scripts/serve_web.py，再用浏览器打开本站 /web/ 下页面（勿 file://）。

仅供学习演示，不构成投资建议。

模型接入

YAML MERGE

写入 configs/arena_config.yaml 的 providers:；再到 擂台配置 拉取配置即可选用。密钥只写在 .env，此处只填变量名。

擂台配置页

选 2～4 个上场槽位、每人指定 provider/模型、一键或单槽生成人格 YAML；导出片段与 arena_run.py --contestants 命令（需 serve_web.py 提供 API）。

擂台竞技页

像素擂台、实时排名、NAV 条形榜；数据来自 arena_state.json（跑完 arena_run.py --mode sim 后刷新）。
